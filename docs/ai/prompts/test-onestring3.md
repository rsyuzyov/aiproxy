Проведи end-to-end тест установочного однострочника из этого репозитория на Proxmox/LXC.

- Подключись по ssh к hv1, там есть контейенр 131
- Пересоздай контейнер на debian 13, но сохрани mac-адрес (шаблон в /pool2/template/cache/)
- В контейнере запусти установку строго однострочником из README.md

Ожидаемый результат:

- Срипт должен завершиться без ошибок (если ошибки, то проверять остальное не требуется)
- Служба cliproxy-api работает
- proxybridge работает
- firefox установлен
- url http://127.0.0.1:8317/management.hrml доступен
- переключение раскладок En и Ru работает
- xrdp работает и слушает порт 3389

При ошибке проанализируй причину. Если требуется внести исправление в скрипты, выполни следующее:

- внеси исправления
- сделай commit & sync
- запусти повторное тестирование

Браузер использовать нельзя.
